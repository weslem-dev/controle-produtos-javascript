# Controle de Produtos em JavaScript

Aplicação web simples para controle de produtos, executada diretamente no navegador.

## Funcionalidades
- Adicionar produtos
- Listar produtos
- Remover produtos dinamicamente

## Tecnologias
- HTML
- CSS
- JavaScript

## Objetivo
Projeto desenvolvido para fins acadêmicos, com foco em lógica de programação, manipulação do DOM e eventos.
